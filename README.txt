---------------------------------------------------------------
programa simples de verificacao
---------------------------------------------------------------
Programa criado para verificar se 3 pessoas e suas derivadas alturas
podem usar um brinquedo ou nao.
Variavel pelas alturas, ou seja,
caso seja maior que 1.80 podera usar o brinquedo
caso seja menor que 1.80 nao podera usar o brinquedo
